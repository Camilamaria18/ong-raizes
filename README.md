<h1 style="color:#2E8B57">Projeto ONG Raízes do futuro</h1>


Este projeto consiste no desenvolvimento de um site institucional para a ONG Raízes do futuro, criado com HTML, CSS e JavaScript. O objetivo é apresentar as ações da organização, exibir seus projetos e disponibilizar um formulário para cadastro de voluntários.

Estrutura do projeto

O site é composto por três páginas principais:

<span style="color:#1e90ff">index.html</span> 
Página inicial contendo apresentação da ONG e suas áreas de atuação.

<span style="color:#0ca87d">projetos.html</span>  
Página destinada à exibição dos projetos ativos da instituição.

<span style="color:#d35400">cadastro.html</span>
Página contendo o formulário para cadastro de voluntários e apoiadores.

Além disso, o projeto possui uma pasta chamada assets, responsável por organizar os recursos utilizados no site:

assets/css/style.css para estilização do site

assets/js/script.js para aplicação de máscaras no formulário e validação básica

assets/imagens para armazenamento de imagens, incluindo o logotipo

Tecnologias utilizadas

As tecnologias aplicadas foram:

HTML5

CSS3

JavaScript

<span style="color:#c0392b">Validação do código:</span>

Para garantir a conformidade com os padrões web, todas as páginas foram validadas utilizando o validador oficial W3C.
As capturas de tela das validações foram incluídas no repositório na pasta destinada às imagens de validação.

<span style="color:#c0392b">Objetivo do projeto:</span>

O projeto foi desenvolvido como atividade acadêmica com foco nos seguintes pontos:

Estruturação correta de páginas HTML

Organização de arquivos em pastas

Criação de formulários com campos obrigatórios

Aplicação de máscaras e validações simples em JavaScript

Separação de estilos e scripts em arquivos externos

Envio e organização dos arquivos em um repositório GitHub
